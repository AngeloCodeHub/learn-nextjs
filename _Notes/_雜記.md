# 開發雜記

- 下載官方新專案（不下載套件實體）
```Powershell
pnpm create next-app@latest my-app --skip-install
```
  舊專案轉移
- 分支混亂，nextjs-learn 強制推上
```Bash
git push origin HEAD --force
```
- 匯入筆記
```Powershell
git checkout "hash" ./_Notes ./.vscode ./.gitattributes ./.obsidian ./.gitignore
```
